Projects

Neural Network Pathfinding 2018
The goal of this project was to experiment with the concepts of genetic algorithms and neural networks . In this project I programmed an algorithm which saught to find the quickest path to a goal, while avoiding user-drwan obsticals. This program would iteratively create populations of neural networks, test their ability, then rank them and create a new population weighted towards the more sucessful individuals. Each individual is fed information about how far it can travel in a cardinal direction before hitting a boundary. Then, this information is processed through its' neural network and the individual chooses a direction to travel in. I experimented with different modes of movement, from traveling in a locked cardinal direction, traveling in any 2d euclidian direction at a fixed rate, traveling with speed depending on output, and finally giving each individual momentum and taking the output as a force instead of the exact direction of travel. Each of these iterations of travel provided interesting results.

MNIST Handwriting 2019
The goal of this project was to experiment with the concepts of supervised learning and backpropagation . In this project I used Python without any machine learning libraries. From scratch, I programmed a neural network that learned to interpret human handwriting. The MNIST dataset was used to train the neural network. In this project I also expierimented with different shapes of neural networks and analaysed how those shapes affected training time and accuracy.
